Porkchat

Some scripts for Stack Exchange chat created by fredley.

Dark Theme

A light-on-dark theme for all chat windows.

Friday Checking

Flag links that are potentially links to 'Friday' by Rebecca Black.

Message Burning

Add a small button next to Star/Flag/Reply to 'burn' a message - hiding it.

Drag+Drop Image Uploading

Drag-and-drop images onto the chat window to upload them.

More Stars

Make sidebar panels collapsible so that there is more room for the star list.

HamStare

Add a menu with buttons for quick-text shortcuts.

PONIES

Fade out images and YouTube previews to 0.1 opacity unless they are moused over.

Topic Diff

Shows a diff message when the topic is changed.
